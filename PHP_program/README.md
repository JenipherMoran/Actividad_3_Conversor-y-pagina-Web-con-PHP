Pàgina Web Hola Mundo con PHP

Este proyecto consiste en una pàgina web sencilla desarrollada en PHP que muestra un mensaje de bienvenida y una imagen generada con inteligencia artificial. El objetivo es comprender còmo ejecutar un servidor local PHP.

Pre-requisitos Antes de ejecutar el proyecto se debe tener instalado: PHP Git Visual Studio Code

Tecnologìa utilizada: PHP

Pasos de ejecuciòn:

Clonar repositorio Entrar a la carpeta del repositorio Ejecutar el servidor PHP: php -S localhost:8000 Abrir en el navegador: http://localhost:8000/
